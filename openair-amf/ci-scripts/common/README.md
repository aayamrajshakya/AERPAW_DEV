------------------------------------------------------------------------------

                             OPENAIR-CN-5G
 An implementation of the 5G Core network by the OpenAirInterface community.

------------------------------------------------------------------------------

OPENAIR-CN-5G is an implementation of the 3GPP specifications for the 5G Core Network.

It contains multiple network elements that are each implemented in its own GIT repository.

This repository is meant for Continuous Integration scripts that are shared by all the network functions.

It may also contain scripts to enable automation in order to push new commits on this shared CI sub-module to all its parent repositories.
